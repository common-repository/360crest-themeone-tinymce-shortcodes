== Description ==

**Foreigncodes Tinymce Shortcodes, is a wordpress tinymce addon, that jazz up your wordpress post with cool design.**

This plugin comes with over 10 preset button styles, toggles, accordion and grid layout you can easily input in your article.

Simple to use / understand feature, loads of options to select from

== Installation ==

From within WordPress

1. Visit **Plugins > Add New**
1. Search for **Foreigncodes Tinymce Shortcodes**
1. Click the **Install Now** button to install the plugin
1. Click the **Activate** button to activate the plugin

Manually

1. Unzip the download package
1. Upload `Foreigncodes-tinymce-shortcodes` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
